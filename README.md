# Smart Library System 

A modern, full-stack web application designed to streamline library management tasks, featuring AI-powered assistance via the Groq API.

## Tech Stack
* **Frontend:** HTML5, Tailwind CSS, JavaScript, React
* **Backend:** Java (Spring Boot)
* **Database:** MariaDB
* **AI Integration:** Groq API

## Key Features
* **User & Admin Dashboards:** Dedicated interfaces for members and administrators.
* **AI Assistance:** Intelligent library search and support powered by Groq AI.
* **Responsive Design:** Optimized for all devices.

## Getting Started
1. **Clone the repository:**
   `git clone https://github.com/dulakshijayasundara08-blip/smart-Library.git`
2. **Run the application:**
   `./mvnw spring-boot:run`
